# BlockSwap Node

This crate contains the BlockSwap node implementation.

# BlockSwap Node (Rust)
## Build
- `cargo build`

## Run
- `cargo run`
## Layout
- `blockchain/` core types (block, tx, state, chain)
- `network/` libp2p stubs
- `storage/` RocksDB init
- `mempool/`, `vm/`, `swap/`, `crosschain/`, `crypto/` placeholders
